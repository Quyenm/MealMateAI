# Kitchen 3D models

Thả các file **.glb** (binary glTF) vào đúng thư mục này (`public/models/`) với ĐÚNG tên dưới đây.
App sẽ load qua `/models/<tên>.glb`. Ưu tiên license **CC0** (khỏi ghi nguồn).

| File cần | Dùng cho | Gợi ý từ khoá tìm |
|---|---|---|
| `tomato.glb`   | cà chua        | tomato |
| `egg.glb`      | trứng          | egg |
| `pan.glb`      | chảo           | frying pan / pan / skillet |
| `plate.glb`    | đĩa            | plate / dish |
| `garlic.glb`   | tỏi            | garlic |
| `scallion.glb` | hành lá        | green onion / scallion |
| `pork.glb`     | thịt           | meat / pork / steak |
| `chili.glb`    | ớt             | chili / pepper |
| `onion.glb`    | hành tím (tuỳ) | onion / shallot |

Chỉ cần vài file cũng được (cà chua + trứng + chảo + đĩa là đủ cho món trứng chiên cà chua).
Tải xong báo Claude tên file đã có → sẽ wire vào scene (tự chỉnh scale/xoay/vị trí cho từng model).

## Nguồn tải (free)

- GualtierisGG low-poly food (CC0): https://gualtierisgg.itch.io/low-poly-food
